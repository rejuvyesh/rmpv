rmpv
====

Ruby wrapper around [mpv](http://mpv.io/) with [trakt](http://trakt.tv) and [myanimelist](http://myanimelist.net) scrobble support.

## Installation

Add this line to your application's Gemfile:

    gem 'rmpv'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rmpv
